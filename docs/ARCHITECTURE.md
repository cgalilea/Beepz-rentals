# System Architecture

## Tech Stack

Frontend
- Next.js
- TypeScript

Backend
- Node.js
- Express

Database
- PostgreSQL

ORM
- Prisma

Testing
- Vitest

## Repo Structure

apps/
  api/ → backend API
  web/ → frontend dashboard + client portal

docs/
  PRD/ → feature specifications

## Development Strategy

The application is built feature-by-feature.

Each feature must include:

- database schema
- API endpoints
- UI components
- automated tests
